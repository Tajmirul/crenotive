
# CRENOTIVE

**Crenotive** is perfect website template for corporate world. [**Demo**](https://crenotive.netlify.app). It's fast, developer friendly template with reuseable code.

### Requirements
Node v14.1.1 or higher 
npm or yarn

### Install packages
Via npm ``npm install``
Via yarn ``yarn``
